# Ejemplo 2: Trabajo con Ramas

Este archivo demuestra cómo trabajar con ramas en Git.

## Instrucciones

1. Crea una nueva rama: `git checkout -b mi-rama-practica`
2. Modifica este archivo agregando tu ejercicio favorito
3. Realiza un commit en tu nueva rama
4. Vuelve a main: `git checkout main`
5. Fusiona tu rama: `git merge mi-rama-practica`
